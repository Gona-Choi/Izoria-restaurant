# Izoria Restaurant Website

A responsive Italian restaurant website built as a 3-person team project.  
🏆 Won 1st place in the final team competition (August 2025)

## Live Demo

https://amazing-pavlova-c3180c.netlify.app/

## Features

- **7-Page Website**: Menu browsing, event listings, and job applications
- **Responsive Design**: Fully responsive layout for mobile, tablet, and desktop
- **Job Application Form**: Recruit page with form validation
- **FAQ Page**: Customer inquiry section
- **Semantic HTML**: Clean, accessible markup
- **Git Workflow**: Branching strategy and code review process

## Tech Stack

- **HTML5**: Semantic markup
- **CSS3**: Custom styling and animations
- **JavaScript**: Interactive functionality
- **Bootstrap 5.3**: Responsive layout and components
